# IPL Analysis (2008–2025) | Interactive Dashboard using Power BI
## Project Objective
The objective of this project is to analyze Indian Premier League (IPL) data from 2008 to 2025 and create an interactive dashboard that provides meaningful insights about team performance, player achievements, tournament statistics, and season-wise results.
This dashboard helps users understand IPL trends such as championship winners, runner-ups, top-performing teams, batting records, bowling records, and overall tournament performance.
