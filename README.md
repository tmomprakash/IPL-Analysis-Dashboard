# IPL Analysis (2008–2025) | Interactive Dashboard using Power BI
## Project Objective
The objective of this project is to analyze Indian Premier League (IPL) data from 2008 to 2025 and create an interactive dashboard that provides meaningful insights about team performance, player achievements, tournament statistics, and season-wise results.
This dashboard helps users understand IPL trends such as championship winners, runner-ups, top-performing teams, batting records, bowling records, and overall tournament performance.

## Questions (KPIs)
- Who won the IPL title in each season?
- Which team finished as runner-up in every season?
- How many total sixes were hit in a selected season?
- How many total fours were scored?
- Total number of matches played in each season?
- Total number of teams participated?
- Which player scored the most runs (Orange Cap holder)?
- Which bowler took the highest wickets (Purple Cap holder)?
- Which team topped the points table?
- How many centuries were scored in a season?
- How many half-centuries were scored?
- How many venues hosted IPL matches?

- Dashboard Interaction <a href="https://github.com/tmomprakash/IPL-Analysis-Dashboard/blob/main/Dashboard_Image.png">View Dashboard</a>

## Process
- Collected IPL historical datasets from reliable sources.
- Cleaned and transformed raw data using Power Query.
- Removed missing values and corrected inconsistent records.
- Created relationships between multiple datasets in Power BI.
- Built DAX measures for KPI calculations.
- Designed an interactive dashboard with slicers for season filtering.
- Added visualizations for points table, champions data, and player statistics.
